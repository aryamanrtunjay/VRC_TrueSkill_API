üìã Available Endpoints

Get Team Skills Data

GET /api/skills?team={team}

Returns skills data for a specific team including skill score, driver score, and programming score.

Example: /api/skills?team=1234A

Get Team TrueSkill Data

GET /api/trueskill?team={team}

Returns TrueSkill ratings, rankings, OPR, DPR, CCVM, and win percentage for a team.

Example: /api/trueskill?team=1234A

Get Skills Leaderboard

GET /api/leaderboard?limit=50

Returns the top teams ranked by skills score. Optional limit parameter (default: 50).

Example: /api/leaderboard?limit=25

API Information

GET /api/hello

Returns API documentation and status in JSON format.

Example: /api/hello

üìä Usage

All endpoints support CORS and return JSON responses. Use query parameters like ?team=1234A for team-specific data.
